Population Machine Learning Models


Objective

Throughout the course of our project, our team created and trained several Machine Learning models. We aimed to predict a country’s future population issues (i.e over/under crowding) to identify the timelines of these problems and prepare for them.

How?

We utilized Machine Learning models and the relationships between different population factors (median age, urban population, etc.) to train the models and create accurate predictions.

View Our Models

Gaussian Regression Model

We utilized the Gaussian Regression Model since it works well with smaller datasets. The 95% confidence interval displayed on the graph demonstrates how certain the machine learning is of its predictions. Below, you can train this model using any two factors from a country.

  • Standard Deviation: 0.5602876245485191
  • MAE: 0.6599639068564018
  • Score: 0.99












Lasso Regression Model

The Lasso Regression Model is a modification of a Linear Regression Model. Therefore, we chose to use this model due to the strong correlations in our data. This model was able to accurately predict data points, such as future population. Below, you can use our model to predict the future population of a country in a certain year.

  • Standard Deviation: 5.876559420041964
  • MAE: 0.2511903017364201
  • Score: 1.0








Support Vector Regression (SVR) Model

The SVR works well with smaller datasets, as well as data in a similar range-this means that it worked well with our population data. Not only is the SVR easy to implement, it also is adept at generalizing and making predictions.

  • Standard Deviation: 5.876559420041964
  • MAE: 3.081251849981352
  • Score: 0.43

Polynomial Regression Model (Africa Fertility Rate)

We used a polynomial regression model to predict fertility rate for Africa, since that would not follow a linear path. This model was able to accurately predict future fertility rates for Africa until 2060, though its predictions were less accurate past that point since the function is not a perfect polynomial graph. Other models could more effectively predict further into the future.
